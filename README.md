## 📌 Overview

SecureShare is a secure file-sharing web application built using Python Flask. It encrypts uploaded files using AES-256 encryption before storing them on the server. Users can share encrypted files using secure links with optional password protection, expiration time, and download limits.

---

## 🚀 Features

- AES-256 File Encryption
- Password-Protected Downloads
- Expiring Download Links
- Maximum Download Limit
- Secure Dashboard
- File Upload &amp; Download
- Automatic Expired File Cleanup
- Responsive UI

---

## 🛠 Technologies Used

- Python

- SQLite
